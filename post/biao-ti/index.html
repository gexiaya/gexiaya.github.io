小白必学！Markdown 语法（从入门到入土）

发布于 2021-12-09 | 6分钟 | 1566字数 | 浏览量：

Markdown是一种轻量级标记语言。其根据使用平台不同，功能和用法会存在些许差异，如果你在使用中感觉不对，别问，问就是不支持。
但是在Gridea里，肯定是为了方便我们撰写博客啊。虽然有编辑器，但这都算旁门左道，你掌握了一种编辑器的食用方法，万一别的平台不认呢？而且这门语法本身也不是很难，我们学在手里，何乐而不为呢？
废话不多说，我们进入教程~~~~~~

一，基本语法--强调元素（字体的倾斜与加粗）

  • 这个元素可以使用两种符号，*和_（下划线），效果和用法都是一样的，我就用*演示了。
    *文字* 代表 斜体
    **文字** 代表 粗体
    ***文字*** 代表 我都要
  • 额外补习一个：
    ~~文字~~ 代表 给文字添加删除线

二，基本语法--标题元素

标题共分为六种格式（没有七号标题，只有h1--h6)：

  • 食用方法（最后一个 # 与文字之间添加一个空格，不然没效果）：
    # 文字 为一号标题
    ## 文字 为二号标题
    ### 文字 为三号标题
    #### 文字 为四号标题
    ##### 文字 为五号标题
    ###### 文字 为六号标题

三，基本语法--区块元素

区块元素表示将一段内容整体融合，使其具有视觉分割性
这个元素只用到一个符号 >
具体的用法为：

  • 不第后赋菊~~黄巢
    （此处打一个 >加空格，然后把下面的诗句复制进去就好啦。）
    待到秋来九月八，
    我花开后百花杀。
    冲天香阵透长安，
    满城尽带黄金甲。
    具体显示效果如下：
  • 不第后赋菊~~黄巢

待到秋来九月八，
我花开后百花杀。
冲天香阵透长安，
满城尽带黄金甲。

  • （值得一提的是，Gridead的编辑器想要实现这种效果，只需要在每句话前面都加上“>加空格”，不过显示效果就有些差强人意了。）

###五，基本语法--代码区块
代码区块就是为代码进行安排独立的区域块，根据不同的编程语言显示不同的高亮

  • 我们所用的符号是```加空格（这三个点是键盘上数字1旁边的那个按键，我们要在英文的状态下点击三次,然后回车，将代码复制进去，在右下角的框中输入我们的编程语言就可以啦。）

  • 正常输入：

java语言：

int a = 10;
int b = 20;
int sum = a + b;
System.out.println

python语言：

num1 = 10;
num2 = 20;
sum = sum1 + sum2;
print(sun);

c语言：

\int num1 = 10;
int num2 = 20；
int sun = sum1 + sum2;
printf(sum)；

加入代码区块的显示效果：

java语言：

int a = 10;
int b = 20;
int sum = a + b;
System.out.println

python语言：

num1 = 10;
num2 = 20;
sum = sum1 + sum2;
print(sun);

c语言：

\int num1 = 10;
int num2 = 20；
int sun = sum1 + sum2;
printf(sum)；
  • （有意思的是，这种输入方式在Grida中也不支持，但我找到了它所支持的，放张图片你自己感受一下，应该不难理解。)

七，高级语法--超链接

1.行内式（常用）
格式：[超链接的名字](具体的url'网址')
输入方法：[阁下的博客](https://blog.gexia.tk)
显示效果：阁下的博客
是不是很简单？
2.参考式（用于网址在文章中需要频繁使用时，节省出手敲网址的时间）
格式：
第一步：起一个名字表示一个网址
[别名]：具体的地址
第二步：使用刚刚起的名字
[使用的地方][别名]
输入方式：
[查阅]:https://www.baidu.com
上面这条为你的参考网址以及它的姓名，下次你要用的时候直接写他姓名就可以了。
我想[吃饭][查阅]啊啊啊啊啊啊啊啊啊啊啊啊
我想[睡觉][查阅]啊啊啊啊啊啊啊啊啊啊啊啊
我想[打游戏][查阅]啊啊啊啊啊啊啊啊啊啊啊
